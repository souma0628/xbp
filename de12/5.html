1. Gitとは何か

Gitはソースコードやファイルの変更履歴を記録し、複数人で安全に開発を進めるための分散型バージョン管理システムです。

リポジトリのイメージ

リポジトリ（repository）は、すべての変更履歴が入っている“プロジェクトのデータベース”のようなもの。ローカル（自分のPC）とリモート（GitHubなど）に存在します。

主なコマンド

  • git init：新しいリポジトリを作成
  • git add .：変更をステージング
  • git commit -m "コメント"：履歴を記録
  • git push origin main：リモートへ送信

よくある疑問

  • 「commit」と「push」の違いは？ → commitはローカルの履歴保存、pushはサーバー（GitHub）に反映。
  • 「branch」とは？ → 作業を並行して進めるための分岐線。mainに統合する前の作業用。

2. HTMLとは何か

HTMLはWebページを構成するためのマークアップ言語です。タグ（<tag>）で構造を指定します。

基本構造

<!DOCTYPE html>
<html>
  <head>
    <title>ページタイトル</title>
  </head>
  <body>
    <h1>見出し</h1>
    <p>段落</p>
  </body>
</html>

3. GitHub Pagesで公開する手順

HTMLファイルを世界に公開する方法を、初心者向けにスクリーンショット付きで説明します。

① GitHubで新しいリポジトリを作成

GitHubにログイン → 右上の ＋ New repository をクリックし、my-website などの名前でリポジトリを作ります。

② index.html をアップロード

リポジトリ画面で Add file → Upload files を選び、自分の index.html をアップロードします。

③ GitHub Pages を有効化

リポジトリの Settings → Pages に進み、Deploy from a branch を選択。

④ ブランチを main / ルート に設定

main ブランチ、フォルダは /(root) を選び、「Save」。

⑤ 公開URLを確認

設定後しばらく待つと、「Your site is live at...」というURLが表示されます。これが自分のWebページです！
